# Boat Image Classification (CNN)

This is a beginner-level deep learning project where I built a Convolutional Neural Network (CNN) to classify different types of boats using images.

## What I did
- Loaded and processed image dataset using ImageDataGenerator
- Applied data augmentation
- Built and trained a CNN model using TensorFlow/Keras
- Evaluated model performance using accuracy and loss

## Tools Used
- Python
- TensorFlow / Keras
- Google Colab

## Project Goal
To understand how image classification works using deep learning.

## Status
Still learning and improving the model performance.
